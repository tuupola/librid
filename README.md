# [WIP] librid
An ASTM F3411 message generating and parsing library written in c.

## Build examples

```
$ mkdir build && cd build
$ cmake -DRID_BUILD_EXAMPLES=ON ..
$ make
```

## Build and run tests

```
$ mkdir build && cd build
$ cmake -DRID_BUILD_TESTS=ON ..
$ make
$ ctest
```
